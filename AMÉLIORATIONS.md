## 🧪 **COMMENT TESTER LES NOUVELLES FONCTIONNALITÉS**

### 1. **Mode Sombre/Clair**
- Cliquez sur 🌙/☀️ en haut à droite
- Ou utilisez `Ctrl + T`
- Vérifiez que tous les éléments s'adaptent

### 2. **Logo Placeholder**
- Le logo Victory est visible en haut à gauche
- Survol pour voir l'animation
- Consultez `LOGO-GUIDE.md` pour personnaliser

### 3. **Raccourcis Clavier**
- `Alt + 1,2,3,4` : Navigation rapide
- `Ctrl + Enter` : Ouvrir panier (sur menu)
- `Ctrl + T` : Basculer thème
- `Échap` : Fermer modales

### 4. **Animations**
- Scroll sur la page d'accueil (cartes qui apparaissent)
- Navigation entre sections (transition fluide)
- Ajout au panier (feedback visuel amélioré)
- Hover sur tous les éléments interactifs

### 5. **Mobile**
- Redimensionnez la fenêtre < 768px
- Navigation mobile avec icônes
- Test touch sur tous les boutons

## 🎨 **PERSONNALISATION RAPIDE**

### 🏷️ **Changer le Logo**
1. Suivez le guide dans `LOGO-GUIDE.md`
2. Remplacez le SVG ou ajoutez votre image
3. Testez en mode sombre et clair

### 🎨 **Changer les Couleurs**
Dans `styles.css`, modifiez les variables CSS :
```css
:root {
    --primary-color: #ff6b00;    /* Votre couleur principale */
    --primary-hover: #ff8c00;    /* Couleur survol */
}
```

### 📝 **Modifier le Contenu**
- **Menu** : Éditer `index.html` section menu
- **Textes** : Rechercher et remplacer dans `index.html`
- **Prix** : Modifier dans les fonctions `addToCart()`
