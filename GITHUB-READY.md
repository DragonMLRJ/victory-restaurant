## 🎯 **GUIDE DE LANCEMENT EN 3 ÉTAPES**

### **ÉTAPE 1 : Sécurisation** ⚠️ **OBLIGATOIRE**
```bash
1. Double-cliquez sur "github-deploy.bat"
2. Choisissez option "4 - Sécurité admin"
3. Modifiez auth.js avec vos vrais identifiants
4. Testez en local avec start.bat
```

### **ÉTAPE 2 : Déploiement GitHub** 🐙
```bash
1. Double-cliquez sur "github-deploy.bat"
2. Choisissez option "1 - Premier déploiement"
3. Suivez le guide pas-à-pas
4. Votre site sera en ligne en 10 minutes !
```

### **ÉTAPE 3 : Configuration Firebase** 🔥 **OPTIONNEL**
```bash
1. github-deploy.bat > option "3 - Firebase"
2. Créez projet Firebase (gratuit)
3. Copiez la configuration dans firebase-db.js
4. Décommentez scripts Firebase dans HTML
5. Redéployez avec option "2 - Mise à jour"
```
